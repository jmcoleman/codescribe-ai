üîë Key Implementation Differences

Shadow Opacity

Light mode uses 20% opacity for subtle depth. Dark mode uses 30% for stronger glow effects.

Color Brightness

Dark mode uses lighter shades (400-series) for better visibility on dark backgrounds. Light mode uses darker shades (600-700 series).

Text Contrast

Both themes meet WCAG AAA standards (7:1+ contrast). Dark mode inverts the slate scale for text.

Cyan Accent

Added in v2.4.5 for "code = cyan" consistency across both themes. Creates terminal-inspired aesthetic.
