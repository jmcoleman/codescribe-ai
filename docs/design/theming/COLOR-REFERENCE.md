## üîç Unique to Each Theme

### Light Mode Only
- `white` (`#FFFFFF`) - Surface color (no dark equivalent)
- Uses 600-700 series for brand colors (darker)
- 20% shadow opacity

### Dark Mode Only
- `slate-950` (`#020617`) - Deepest background (no light equivalent)
- Uses 400-500 series for brand colors (lighter)
- 30% shadow opacity for stronger glow

## üí° Usage Tips

1. **Brand Colors:** Purple (primary), Indigo (secondary) - use lighter shades in dark mode
2. **Cyan = Code:** Consistent semantic meaning across both themes
3. **Shadows:** 20% (light) vs 30% (dark) opacity for depth
4. **Text:** Invert slate scale (900‚Üí100) when switching themes
5. **Semantic:** Use 400-series in dark mode for better visibility
