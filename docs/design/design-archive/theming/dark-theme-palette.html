Built for developers who code at night. This dark theme maintains CodeScribe's brand identity while providing exceptional contrast, reduced eye strain, and a sleek modern aesthetic. We've carefully balanced vibrant accent colors against rich dark surfaces to create a professional yet inviting coding environment.

‚ú® Design Highlights

  • Deep Slate Base: Rich dark backgrounds (slate-950, slate-900) for maximum depth
  • Elevated Surfaces: Layered UI with subtle transparency and borders
  • Vibrant Accents: Lighter purple/indigo shades (400-500) pop against dark backgrounds
  • Enhanced Contrast: All text meets WCAG AAA standards on dark backgrounds

  • Reduced Blue Light: Warm undertones reduce eye strain during long coding sessions

üìã Dark Theme Usage Guidelines

üé® Surface Layers

  • Base: slate-950 (app background)
  • Elevated: slate-900 (panels, modals)
  • Cards: slate-800 (code editor, cards)
  • Hover: slate-700 (interactive states)
  • Borders: slate-600 with 10% opacity

üü£ Purple (Primary)

  • Buttons: purple-400 background
  • Hover: purple-500
  • Links: purple-400
  • Focus rings: purple-500
  • Glows: purple-400 with 30% opacity

üîµ Indigo (Secondary)

  • Secondary buttons: indigo-400
  • Badges: indigo-400 with 20% bg
  • Technical elements: indigo-500
  • Code blocks: indigo-400/10 bg

üíé Cyan (Accent)

  • Syntax highlighting (variables)
  • Special state indicators
  • Neon glow effects
  • Active/selected states
  • Terminal-style accents

üìù Text Hierarchy

  • Headings: slate-100 (brightest)
  • Body: slate-200
  • Secondary: slate-300
  • Muted: slate-400
  • Disabled: slate-500

‚úÖ Semantic States

  • Success: green-400 (brighter in dark)
  • Warning: amber-400 (warm, visible)
  • Error: red-400 (high contrast)
  • Backgrounds: Use 15% opacity
  • All meet WCAG AAA standards
