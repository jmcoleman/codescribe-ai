  • Two modes: Cloud (default) and Local (--local flag)
  • Cloud mode: OAuth login flow (opens browser, saves JWT token)

  • Local mode: User provides own Claude API key (bypasses platform)

  • File path support (single file or directory)
  • Batch processing with glob patterns
  • Configuration file support (.codescriberc)
  • Cross-platform support (Windows, macOS, Linux)

  • CI/CD integration (GitHub Actions, GitLab CI, Jenkins)
